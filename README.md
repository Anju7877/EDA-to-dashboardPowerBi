# EDA to Dashboard - Power BI

## Project Overview

This project demonstrates how Exploratory Data Analysis (EDA) can be transformed into interactive Power BI dashboards for effective business decision-making and data storytelling.

## Objective

* Convert raw data insights into interactive dashboards.
* Present business metrics in a visual format.
* Support data-driven decision-making.

## Tools Used

* Power BI
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Excel

## Dashboard Features

* Interactive Filters and Slicers
* KPI Monitoring
* Trend Analysis
* Data Visualization
* Business Insights Dashboard
